# Bash скрипты

Набор Bash скриптов для выполнения различных задач.

## Список скриптов

- greet.sh - скрипт приветствия
- sum.sh - калькулятор суммы
- even_odd.sh - проверка числа на чётность
- create_pr.sh - создание структуры проекта
- lines.sh - подсчёт количества строк в файле
- password_generator.sh - генерация случайного пароля 
- extension.sh - поиск файлов по расширению
